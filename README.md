# onlineShop
A simple online Shop 

## Dependencies
install Dependencies by :-
```bash
npm init
```
## Setup database
 For windows :-
 ```bash
mongod
```

For Linux:-
```bash
service mongodb start
```

## Run app

```bash
npm start
```
```
http://localhost:3000/
```
[visit](https://online-shop-in.herokuapp.com/)
